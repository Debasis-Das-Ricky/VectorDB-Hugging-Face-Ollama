# VectorDB-Hugging-Face-Ollama
VectorDB + HF + Ollama RAG Pipeline - Local AI chatbot using ChromaDB/Chroma for semantic search on documents, Hugging Face sentence-transformers for fast embeddings, and Ollama Llama3 for grounded generation. Perfect for private RAG apps, medical research QA, data science projects. PDF loader + Streamlit UI included. Runs 100% offline!
